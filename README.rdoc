== ETSY-APP

=== What is it?
This is a clone of the Etsy site, following along with the Codeacademy tutorial

=== What you need on your computer
Ruby & Ruby on Rails

=== How to set up and run 

  $ git clone git@github.com:sandagolcea/etsy-app.git
  $ cd etsy-app
  $ rails server
open your browser and go http://localhost:3000 or to http://localhost:3000/categories
